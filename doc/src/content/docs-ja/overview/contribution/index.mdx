このプロジェクトは、Tailwind の生の数値ユーティリティの代わりにセマンティックなデザイントークンを強制するリンターです。もともと、自分のプロジェクトで一貫したデザイントークンの使用を強制したくて作り始めました。最初はあるプロジェクトのサブパッケージとして始まりましたが、すべてのプロジェクトでドキュメントツールを使っているため、パッケージをプロジェクトディレクトリにコピー&ペーストし続けることになりました。そこで、一貫性と保守のしやすさのために npm に公開しました。

このプロジェクトのソースコードの大部分は [Claude Code](https://claude.ai/) によって書かれています。オープンソースプロジェクトとして公開していますが、今後積極的にメンテナンスを続けるかどうかは未定です。フォークしたりプルリクエストを送ったりすることは自由ですが、レビューやマージを保証することはできません。

すでに AI の時代ですので、リポジトリをフォークして自分で必要な修正を加えることをお勧めします。
