First stable release of `@takazudo/zudo-design-token-lint`. The CLI, the programmatic API, and the `.design-token-lint.json` configuration format are now considered stable and follow semantic versioning.

- CLI, programmatic API (`lintFile`, `lintContent`, `checkClass`, `extractClasses`, and the config-loading utilities), and the `.design-token-lint.json` config format are now stable
- No breaking changes from v0.2.0 — this release promotes the existing surface to a 1.0 stability guarantee
- Future breaking changes will bump the major version, following semver
- Bleeding-edge builds are now published under the `next` dist-tag alongside the stable `latest` tag
